Questoes

O que é Justify-Content?

A propriedade CSS `justify-content` define como os itens flexíveis ou de grade são alinhados ao longo do eixo principal do seu contêiner. Ela controla a distribuição do espaço entre e ao redor do conteúdo, com valores comuns como `center`, `flex-start`, `flex-end`, `space-between`, `space-around` e `space-evenly`. É essencial para o design de layouts responsivos.

O que é Display Flex?

O display: flex é uma propriedade CSS que ativa o módulo Flexbox, transformando um elemento em um contêiner flexível. Isso alinha automaticamente seus filhos diretos em uma linha (ou coluna), facilitando a distribuição de espaço, alinhamento (horizontal/vertical) e a criação de layouts responsivos, sendo ideal para componentes unidimensionais.

O que é Align-Items?

A propriedade CSS `align-items` define como os itens flexíveis ou de grade são alinhados ao longo do eixo transversal (perpendicular ao eixo principal). Ela é usada para alinhar itens verticalmente em um layout baseado em linhas ou horizontalmente em um layout baseado em colunas, com valores comuns incluindo `stretch` (padrão), `center`, `flex-start`, `flex-end` e `baseline`.
